# BIM Chatbot Knowledge Base

Dit zijn de kennisbestanden voor de BIM Chatbot van Inholland Alkmaar.

## Bestanden

| Bestand | Inhoud |
|---------|--------|
| `curriculum.json` | Vakken per jaar/periode |
| `boekenlijst.json` | Verplichte boeken 2025-2026 |

## Gebruik

De chatbot fetcht deze bestanden automatisch van:
```
https://raw.githubusercontent.com/JOUW_USERNAME/bim-chatbot-knowledge/main/
```

## Updaten

1. Edit de JSON bestanden
2. Commit & push naar GitHub
3. Chatbot gebruikt automatisch de nieuwe data
